Children

There are two interesting methods: pager.activePage$ that is an observable pager.Page to the active (leaf) child; pager.Page.prototype.child that returns an observable to a child page by name. 

// currentlyActivePage is a pager.Page
var currentlyActivePage = pager.page.activePage$();
// startPage is an observable pager.Page
var startPage = currentlyActivePage.child('start');
